                   <h1>Python</h1>
                   <p>1. High-level, interpreted language.<br>
                      2. Easy syntax — very close to English.<br>
                      3. Popular for AI, data science, web development.<br>
                      4. Large library support (NumPy, Pandas, TensorFlow).<br>
                      5. Cross-platform and open-source. </p>
                   </div>
 
                   <div class="B">
                   <h1>C</h1>
                   <p>1. Oldest and most powerful programming language.<br>
                      2. Used for system programming and operating systems.<br>
                      3. Very fast and efficient (close to hardware).<br>
                      4. Supports structured programming with pointers.<br>
                      5. Basis for many other languages like C++ and C#. </p>
                   </div>

                   <div class="C">
                   <h1>Java</h1>
                   <p>1. Object-oriented, platform-independent language.<br>
                      2. Runs on JVM (Java Virtual Machine).<br>
                      3. Widely used in enterprise applications and Android apps.<br>
                      4. Strong memory management with garbage collection.<br>
                      5. Large community and frameworks (Spring, Hibernate). </p>
                   </div>
            
                   <div class="D">
                   <h1>Cursor</h1>
                   <p>1. A modern AI-powered code editor.<br>
                      2. Built on top of Visual Studio Code.<br>
                      3. Provides AI-assisted coding, debugging, and refactoring.<br>
                      4. Supports multiple programming languages.<br>
                      5. Helps boost productivity for developers.</p>
                   </div>
                  
                   <div class="E">
                   <h1>C++</h1>
                   <p>1. Extension of C → Adds object-oriented programming to C’s features.<br>
                      2. Supports OOP concepts like classes, objects, inheritance, polymorphism.<br>
                      3. Used in system programming (OS, compilers), gaming, and high-performance apps.<br>
                      4. Provides both low-level (memory, pointers) and high-level abstraction.<br>
                      5. Widely used for competitive programming due to speed and STL (Standard Template Library).</p>
